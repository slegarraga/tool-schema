# Maintainers

## Primary maintainer

## Maintainer responsibilities

- Review issues and pull requests.
- Maintain releases and npm package publishing.
- Keep provider schema rules current for OpenAI, Anthropic, Gemini and MCP.
- Triage compatibility, security and documentation reports.

## Scope

`tool-schema` is maintained as a public MIT package for provider-valid
tool/function schemas. The primary maintainer controls the GitHub repository,
npm package, release tags and security policy.
